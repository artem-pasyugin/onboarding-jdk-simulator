## Цикл `while` (#3)

Используя `while` - создайте функцию, которая суммирует предыдущую сумму шагов с текущим шагом от нуля до значения
входящего параметра `number`. Одно условие - шаг равен четному значению.

Например:

- `number` = `2`, ответ `2`.
- `number` = `4`, ответ `6`.
- `number` = `10`, ответ `30`.
