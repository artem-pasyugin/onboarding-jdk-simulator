## Создание классов (#3)

Создайте класс `Robot` и следующие четыре функции:
`right(int steps)`, `left(int steps)`, `down(int steps)` и `up(int steps)`.

Каждая функция должна отображать с консоли следующие фразы:

```text
Right N steps
Left N steps
Down N steps
Up N steps
```

где N это количество шагов.
